# Итоговая работа по курсу "Аналитик данных"

Был проведен анализ продаж магазина, который реализует велосипеды и все, 
что связано с ездой на велосипедах: экипировка, запчасти, аксессуары.

## Были решены следующие задачи:
1) Спрогнозирован объем продаж на следующий квартал (платформа Loginom).
2) Построена модель, предсказывающая вероятность возврата проданного товара (Pycaret).
3) Проведен анализ клиентов (ABC, XYZ, RFM). Составлен список приоритетных клиентов для формирования
персональных предложений. Составлен портрет «идеального» клиента.
4) Проведен анализ ассортимента (ABC, XYZ): по каким товарам необходимо сформировать запас ввиду
их высокого спроса, а какие товары необходимо перевести на систему предзаказ, ввиду их
низкой популярности.
5) Реализован дашборд, отражающий продажи по странам и по периодам.
