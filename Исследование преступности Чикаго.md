## Исследование преступности Чикаго с 2012 по 2017 гг.
#### Цель анализа:
1) Посмотреть как изменилась преступность за эти годы? 
2) Как изменилась преступность по районам? 
3) Какие районы безопаснее? 
4) Можно ли предсказать, где и когда будет совершено следующее преступление? 
5) Для машинного обучения с учителем - предсказать будет арест или нет.

Проведена подготовка и исследовательский анализ данных. 
Построено 8 моделей по датасету, который включает данные о преступлениях в 5 самых криминальный районах Чикаго. 
Целевой показатель - количество преступлений в месяц в определенном районе.

#### Для построения моделей использовались:
1) библиотека ScLearn (методы Random Forest Regressor, Gradient Boosting Regressor, Линейная регрессия);
2) библиотека Pycaret;
3) H2O - node-инструмент для построения моделей;
4) фреймворки машинного обучения CatBoost, XGBM, LightBoost

Лучшие результаты показал H2O с регрессионной моделью, основанной на ансамблевом методе Стэкинг. 
Данная модель объясняет целевой показатель (количество преступлений по видам за месяц в конкретном районе) 
с дисперсией зависимой переменной 0,9017 и абсолютным отклонением в 2,3 преступления - это лучший показатель 
из 8 рассмотренных нами методов.
